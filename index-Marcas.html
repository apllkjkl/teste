Marcas (tags)

HTML element content categories Todo documento HTML possui marcadores (do inglês: tags), palavras entre parênteses angulares (chevron) (< e>); esses marcadores são os comandos de formatação da linguagem. Um elemento é formado por um nome de marcador (tag), atributos, valores e filhos (que podem ser outros elementos ou texto). Os atributos modificam os resultados padrões dos elementos e os valores caracterizam essa mudança.
